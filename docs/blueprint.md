# **App Name**: SpendWise

## Core Features:

- Expense Logging: Users can easily add, edit, and delete expense entries, including amount, date, description, and category.
- Category Management: Supports predefined and user-customizable expense categories (e.g., Food, Transport, Entertainment).
- Spending Overview: Displays daily, weekly, and monthly summaries of all recorded expenses.
- Dashboard Insights: An interactive dashboard presenting basic visual charts (e.g., pie chart for category breakdown, line chart for monthly trend).
- AI Category Suggestion Tool: An AI-powered tool that suggests relevant categories for new expenses based on the description and historical spending patterns.
- Local Data Persistence: Securely stores all expense data locally on the device.
- Ad Monetization: Integrates Google AdMob to display banner advertisements in a non-intrusive manner within the app's UI.

## Style Guidelines:

- Color scheme is light, prioritizing clarity and data readability.
- Primary color: A confident and stable blue (#3366C0), selected for its association with trust and control in financial contexts.
- Background color: A very light, subtle cool-grey (#EFF1F5), offering a clean and modern canvas for content.
- Accent color: A vibrant cyan (#33BDBB), providing visual contrast for interactive elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and highly readable characteristics, suitable for displaying financial data and summaries.
- Uses clean, line-based icons for app functionalities and customizable, visually distinct icons for expense categories.
- Clean and intuitive mobile-first layout with distinct cards or sections for improved scannability and data organization on the dashboard.
- Subtle and smooth transitions for screen navigation and chart loading, providing a fluid user experience without distraction.
